# ProxyOku's Hash Cracking Script (ProxyHCS)

A simple python script to crack a hash using hashlib

## About
My first python script for cybersecurity and ethical hacking purposes. Made over spring break as a learning experience

## Disclaimer
This script is for educational purposes only! Only use it on hashes you have permission to test.

## How to use
1. Ensure you have Python 3.x installed on your system. You can check it by running: `python --version`
2. Clone the repository: `git clone https://github.com/proxyoku/ProxyHCS`
3. Change into the script's directory if needed: 
`cd ProxyHCS`
4. Make sure your target hash and wordlist file are in the same directory as the script. (The name of the text files must match the script)
5. run the script
`python ProxyHCS.py`


## Notes
The text files needed are included for demonstration purposes :)
Works with any hash type that hashlib supports (will have to edit the file depending on hash type)
